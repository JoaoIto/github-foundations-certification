# Introduction to GitHub's Products

> [!NOTE]
> Este módulo aborda os diferentes tipos de contas, planos de faturamento e produtos oferecidos pelo GitHub, ajudando a escolher a melhor opção para você ou para sua organização.

## 1. Tipos de Contas no GitHub

O GitHub possui entidades diferentes para acomodar desde desenvolvedores individuais até grandes corporações.

### Conta Pessoal (Personal Account)
- Todo usuário do GitHub precisa ter uma conta pessoal.
- É a sua identidade na plataforma.
- Pode possuir repositórios (públicos e privados).
- Permite colaborar em repositórios de outros usuários ou organizações.

### Organização (Organization)
- Contas compartilhadas onde grupos de pessoas podem colaborar em diversos projetos ao mesmo tempo.
- Não possuem senhas e não podem "fazer login". São gerenciadas por usuários com contas pessoais que recebem a função de *Owner* (Proprietário) ou *Administrator*.
- Oferecem ferramentas sofisticadas de segurança, administração e gerenciamento de equipes.

### Empresa (Enterprise)
- A conta de nível mais alto, que permite gerenciar múltiplas Organizações sob um único guarda-chuva.
- Centraliza faturamento, políticas de segurança, e permite o uso do *SAML Single Sign-On (SSO)* globalmente.

## 2. Planos de Assinatura (Billing Plans)

O GitHub oferece diversos níveis de serviço, dependendo da necessidade de ferramentas avançadas:

| Plano | Foco principal | Funcionalidades Destacadas |
|-------|----------------|----------------------------|
| **GitHub Free** | Indivíduos e equipes pequenas (Open Source) | Repos públicos e privados ilimitados, Actions e Packages básicos, Issues, Projects. |
| **GitHub Team** | Equipes de desenvolvimento profissional | Tudo do Free + revisores múltiplos, Code Owners, GitHub Pages em repositórios privados, suporte web. |
| **GitHub Enterprise** | Grandes organizações com regras rígidas de segurança | Tudo do Team + SAML SSO, provisionamento SCIM, políticas de compliance avançadas, opções de deploy (Cloud ou Server). |

## 3. GitHub Desktop e GitHub Mobile

Para acessar e interagir com a plataforma além do navegador e do terminal (CLI), existem duas ferramentas oficiais:

- **GitHub Desktop:** Uma interface gráfica de usuário (GUI) para interagir com o Git e o GitHub diretamente do seu computador (Windows/Mac). Excelente para quem não tem familiaridade com comandos de terminal.
- **GitHub Mobile:** Um aplicativo para iOS e Android que permite gerenciar Issues, responder a comentários em Pull Requests e aprovar deploys enquanto você não está no computador. *Aviso: Não é um editor de código.*
