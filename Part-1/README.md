# GitHub Foundations Learning Path - Parte 1

Bem-vindo à **Parte 1** da trilha de aprendizado do GitHub Foundations! 

Nesta primeira metade, o foco é construir a sua base fundamental sobre versionamento de código, a plataforma GitHub e o ecossistema moderno de ferramentas de produtividade. Você começará do zero com comandos Git e evoluirá até automações e ambientes de nuvem.

## 🗂️ Sumário dos Módulos (Parte 1)

Aqui estão os 8 módulos que compõem a base introdutória do seu estudo:

1. [**Introduction to Git**](./01-Introduction-to-Git/README.md)  
   *Aprenda a instalar, configurar e rodar os comandos básicos do Git (add, commit, status).*

2. [**Introduction to GitHub**](./02-Introduction-to-GitHub/README.md)  
   *Conecte seu repositório local à nuvem e aprenda o básico da colaboração na interface do GitHub.*

3. [**Introduction to GitHub's products**](./03-Introduction-to-GitHubs-products/README.md)  
   *Entenda os diferentes planos do GitHub (Free, Pro, Team, Enterprise) e onde usá-los.*

4. [**Configure code scanning**](./04-Configure-code-scanning/README.md)  
   *Introdução a análises automáticas de segurança e ferramentas como o CodeQL.*

5. [**Introduction to GitHub Copilot**](./05-Introduction-to-GitHub-Copilot/README.md)  
   *O que é a inteligência artificial do GitHub, como habilitar e como usá-la de forma responsável.*

6. [**Code with GitHub Codespaces**](./06-Code-with-GitHub-Codespaces/README.md)  
   *Como rodar sua IDE e projetos inteiros em containers na nuvem, sem instalar nada na máquina.*

7. [**Hello GitHub Actions**](./07-Hello-GitHub-Actions/README.md)  
   *Crie o seu primeiro fluxo de automação (CI/CD) para testar e validar código automaticamente.*

8. [**GitHub project management**](./08-GitHub-project-management/README.md)  
   *Organize equipes e entregas usando GitHub Projects, Issues, Milestones e Labels.*

---
> **Dica de Estudo:** Não se esqueça de acessar os arquivos `use-case.md` e `commands.sh` dentro de cada módulo para fixar a teoria com a prática no terminal!
