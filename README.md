Bem-vindo ao repositório definitivo para a certificação **GitHub Foundations**! 🎓

## 🎯 Objetivo deste Repositório

Este repositório foi construído como um portfólio e guia de estudos para a certificação oficial **GitHub Foundations** oferecida pela Microsoft e pelo GitHub. O intuito é centralizar toda a teoria, conceitos-chave, comandos essenciais e cenários práticos necessários para dominar a plataforma, servindo tanto como material de revisão quanto como consulta para o dia a dia.

## 📚 Documentação Oficial
O conteúdo baseia-se nas trilhas de aprendizado oficiais da Microsoft Learn:
- 🔗 [GitHub Foundations - Part 1](https://learn.microsoft.com/en-us/training/paths/github-foundations/)
- 🔗 [GitHub Foundations - Part 2](https://learn.microsoft.com/en-us/training/paths/github-foundations-2/)
- 🔗 [Página da Certificação](https://resources.github.com/learn/certifications/)

## 🗂️ Sumário de Conteúdos

Abaixo está o índice completo de todos os módulos estudados. Para os módulos que envolvem comandos e ferramentas, você encontrará três tipos de arquivos:
1. 📖 `README.md`: A teoria e os conceitos explicados de forma didática.
2. 🛠️ `use-case.md`: O cenário prático que simula o uso no mundo real.
3. 💻 `commands.sh`: Um script executável/documentado contendo os comandos bash com comentários, pronto para ser testado.
