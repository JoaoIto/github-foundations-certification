# GitHub Foundations Learning Path - Parte 2

Bem-vindo à **Parte 2** da trilha de aprendizado!

Após dominar os conceitos básicos, a Parte 2 aprofunda o conhecimento nas rotinas do dia a dia corporativo, melhores práticas de comunidade, segurança e fluxos colaborativos avançados (Pull Requests, Históricos de Commits, Autenticação, etc).

## 🗂️ Sumário dos Módulos (Parte 2)

Aqui estão os módulos focados em consolidação prática e colaboração segura:

1. [**Contribute to an open source project**](./01-Contribute-open-source/README.md)  
   *Como achar projetos abertos, o fluxo de Fork e Pull, e as boas práticas de contribuição comunitária.*

2. [**Manage an InnerSource program**](./02-Manage-InnerSource-program/README.md)  
   *O que é InnerSource (código aberto, mas apenas dentro das paredes da sua empresa) e suas vantagens.*

3. [**Maintain a secure repository**](./03-Maintain-secure-repository/README.md)  
   *Protegendo seu código: Alertas de vulnerabilidades do Dependabot e segredos protegidos.*

4. [**Introduction to GitHub administration**](./04-Introduction-GitHub-administration/README.md)  
   *Permissões, equipes (teams), configurações de segurança no nível organizacional.*

5. [**Authenticate and authorize user identities**](./05-Authenticate-authorize-identities/README.md)  
   *Métodos de 2FA, conexões corporativas com SSO (SAML) e ferramentas de Single Sign-On.*

6. [**Introduction to pull requests**](./06-Introduction-to-pull-requests/README.md)  
   *Anatomia de um PR, tipos de mesclagem (Merge, Squash, Rebase) e resolução de aprovações.*

7. [**Search and organize repository history**](./07-Search-organize-repository-history/README.md)  
   *Uso de buscas globais, `git blame`, cross-linking de issues, e identificação de origem de problemas.*

8. [**Use GitHub Copilot with Python**](./08-Use-GitHub-Copilot/README.md)  
   *Aplicação prática de Inteligência Artificial usando Copilot no Codespaces para criar uma API.*
